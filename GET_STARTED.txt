                         üÜò NEED HELP?
================================================================================

PROBLEM: "Port 5000 already in use"
SOLUTION: lsof -ti:5000 | xargs kill -9
          Then try: ./start.sh

PROBLEM: Can't create admin account
SOLUTION: Make sure you're at http://localhost:5000/setup
          Clear browser cache (Cmd+Shift+Delete)

PROBLEM: Students not getting rooms
SOLUTION: Go to Rooms and make sure rooms exist
          Check room capacity is > 0

PROBLEM: Forgot admin password
SOLUTION: Delete hostel_manager.db
          Restart: ./start.sh
          Create new admin account

PROBLEM: Email reminders not working
SOLUTION: Check email settings (Settings ‚Üí Email)
          Use Gmail App Password (not regular password)
          Enable 2-Step Verification in Google Account
